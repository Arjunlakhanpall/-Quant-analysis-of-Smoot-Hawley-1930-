# -Quant-analysis-of-Smoot-Hawley-1930-
This project tests whether the **Smoot–Hawley Tariff Act** amplified the **Great Depression** through trade collapse and regime change. Using 1925–1935 U.S. data (imports, exports, GDP, industrial production, unemployment, Dow), it applies structural breaks, OLS, event study, and Granger causality in Python
